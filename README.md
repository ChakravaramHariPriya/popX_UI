# 📱 PopX UI Clone

A modern and responsive **PopX UI Clone** built using **React.js**, **Vite**, **React Router DOM**, **Framer Motion**, and **CSS3**. The application recreates the provided Adobe XD design with smooth navigation, responsive layouts, animated transitions, and client-side form validation.

---

## ✨ Features

* 📱 Mobile-first responsive design
* 🎨 Clean and modern user interface
* 🚀 Smooth page transitions using Framer Motion
* 🔐 Login & Signup pages
* 👤 Profile page
* ✅ Form validation before navigation
* 🎯 Reusable React components
* 🌈 Custom gradient theme
* ⚡ Built with Vite for fast performance

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/ChakravaramHariPriya/popX_UI.git
```

Move into the project folder

```bash
cd popX_UI
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build the project

```bash
npm run build
```

## ✅ Form Validation

### Login

* Email is required.
* Password is required.
* Displays an alert if required fields are empty.

### Signup

* Full Name is required.
* Phone Number is required.
* Email Address is required.
* Password is required.
* Prevents navigation until all mandatory fields are completed.
